Postal Code Lookup & City Autocomplete

Two-way address intelligence. Type a postal code — the city and country fill themselves. Start typing a city — a dropdown offers suggestions and fills the postal code back. 4 094 postal codes across Slovakia and Czech Republic, one JSON index, zero external APIs.

This form demands a PHP server to function. Run php -S localhost:8000 in the project folder, then open http://localhost:8000/demo9.html.

Features demonstrated

🔍 lookup — Postal code field triggers an API call after 5 digits, auto-fills city and country
✍ autocomplete_from — City field shows typeahead suggestions with keyboard navigation
↔ map — Selecting a city from autocomplete also fills the postal code and country fields

How it works

  • 🔍 Type 5 digits into Postal Code → lookup fetches api-psc.php?psc=XXXXX → city and country fill automatically
  • ✍ Start typing a city name → autocomplete_from fetches suggestions → dropdown with flags appears
  • ↔ Select a city → map fills postal code and country fields
  • ✓ Submit → all fields validated and stored

The lookup config

"lookup": { "url": "api-psc.php?psc={{value}}", "trigger": 5, "map": { "city": "city", "country": "country" } }

Five digits typed → one fetch → city and country filled.

The autocomplete config

"autocomplete_from": { "url": "api-psc.php?city={{value}}", "min_length": 2, "debounce": 200, "map": { "psc": "psc", "country": "country" } }

Type two letters → dropdown appears → select fills city, postal code, and country.

Data source

4 094 postal codes and 4 043 cities across Slovakia and Czech Republic. Pre-indexed from CSV into two JSON files (~200 KB total). No external API dependencies.

data/psc-to-city.json — postal code → {city, country}
data/city-to-psc.json — city → {postal codes, country}
