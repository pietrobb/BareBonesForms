PSČ Lookup & City Autocomplete

Two-way address intelligence. Type a postal code — the city fills itself. Start typing a city — a dropdown offers suggestions and fills the postal code back. 4 094 PSČ, 4 043 obcí, one JSON index, zero external APIs.

This form demands a PHP server to function. Run php -S localhost:8000 in the project folder, then open http://localhost:8000/demo9.html.

New features demonstrated

🔍 lookup — PSČ field triggers API call after 5 digits, auto-fills city
✍ autocomplete_from — City field shows typeahead suggestions with keyboard navigation
↔ map — Selecting a city from autocomplete also fills the PSČ field

How it works

  • 🔍 Type 5 digits into PSČ → lookup fetches api-psc.php?psc=XXXXX → city field fills automatically
  • ✍ Start typing a city name → autocomplete_from fetches suggestions → dropdown appears
  • ↔ Select a city → map fills the PSČ field with the city's postal code
  • ✓ Submit → all fields validated and stored

The lookup config

"lookup": { "url": "api-psc.php?psc={{value}}", "trigger": 5, "map": { "city": "city" } }

Five digits typed → one fetch → one field filled.

The autocomplete config

"autocomplete_from": { "url": "api-psc.php?city={{value}}", "min_length": 2, "debounce": 200, "map": { "psc": "psc" } }

Type two letters → dropdown appears → select fills city + PSČ.

Data source

4 094 postal codes and 4 043 cities across Slovakia and Czech Republic. Pre-indexed from CSV into two JSON files (~170 KB total). No external API dependencies.

data/psc-to-city.json — PSČ → city
data/city-to-psc.json — city → PSČ(s)
