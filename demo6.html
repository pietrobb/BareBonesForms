Print Shop Order

An order form with live pricing. Select a product, configure variants, choose delivery — the sidebar price summary updates in real time. Demonstrates conditional product groups, variant-based pricing, volume discounts, and conditional delivery options.

This form demands a PHP server to function. Run php -S localhost:8000 in the project folder, then open http://localhost:8000/demo6.html.

What works & what needs setup

✓ Fully functional out of the box. No configuration needed.
📁 Submissions are saved as JSON files in submissions/demo-order/.
✉ Email confirmations require SMTP config in config.php.

Order Summary

Select a product to see pricing.

How pricing works

  • 1️⃣ Base price per piece depends on the product.
  • 2️⃣ Each variant option applies a multiplier (e.g. Premium paper = +50%).
  • 3️⃣ Quantity discount kicks in at 100+ pieces (up to 20% off at 1000+).
  • ✓ Delivery cost depends on destination and speed.

Features demonstrated

  • conditional product groups
  • per-product quantity min/max
  • variant-based price multipliers
  • volume discounts (5%–20%)
  • conditional delivery speed
  • conditional shipping address
  • payment method with COD fee
  • live sidebar price engine
  • hidden field (order total)
  • email confirmations
  • multi-page (2 pages)
  • GDPR consent
