8

Homeopathic Modalities

Select body areas that bother you — for each, the same set of modality questions appears. All generated from a single reusable template. Demonstrates the new templates, use, and prefix properties. One template definition, six instances.

This form demands a PHP server to function. Run php -S localhost:8000 in the project folder, then open http://localhost:8000/demo8.html.

What works & what needs setup

✓ Fully functional out of the box. No configuration needed.
📁 Submissions are saved as JSON files in submissions/demo-modalities/.

How templates work

  • 1️⃣ Define a modalities template once: 5 fields (time, temperature, motion, relief, intensity).
  • 2️⃣ Each body area group uses "use": "modalities" with a unique "prefix".
  • 3️⃣ Engine resolves: time_worse → head_time_worse, stomach_time_worse, etc.
  • ✓ 6 areas × 5 questions = 30 fields from a single 5-field template.

Features demonstrated

  • templates (reusable field sets)
  • use + prefix on groups
  • conditional groups per checkbox
  • multi-page (3 pages)
  • client-side remedy scoring
  • data-hide-on-success
  • server stores the submission

JSON savings

Without templates: 30 fields defined With templates: 5 fields + 6 references Reduction: ~80% less JSON
