The Spanish Inquisition Entrance Exam

A Monty Python–themed quiz with randomized answer order. Every page load shuffles the options, so memorising positions won’t help. Demonstrates the new shuffle property on radio, select, and group fields.

This form demands a PHP server to function. Run php -S localhost:8000 in the project folder, then open http://localhost:8000/demo7.html.

What works & what needs setup

✓ Fully functional out of the box. No configuration needed.
📁 Submissions are saved as JSON files in submissions/demo-quiz/.

How it works

  • 1️⃣ Answer 10 questions across 2 pages of absurd trivia.
  • 2️⃣ Options are shuffled on every page load — no muscle memory!
  • 3️⃣ Submit, get your score, verdict, and see which ones you got wrong.

Features demonstrated

  • shuffle on radio options
  • shuffle on select options
  • multi-page (3 pages)
  • client-side scoring engine
  • post-submit results panel
  • data-hide-on-success
  • server stores the submission
